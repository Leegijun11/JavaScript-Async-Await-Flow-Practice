# JavaScript 비동기 처리 스터디 프로젝트 (Study Project)

---

## 프로젝트 소개

이 프로젝트는 **JavaScript의 비동기 처리 방식(Promise, async/await)** 을 학습하기 위해 제작한 스터디용 예제입니다.

간단한 매점 대화 시나리오를 통해 비동기 함수의 실행 흐름을 단계적으로 구현했습니다.

프로그램 흐름은 다음과 같습니다:

1. 학년 입력
2. 이름 입력
3. 매점 제안
4. 메뉴 선택 (라면 / 김밥)
5. 선택 결과에 따른 메시지 출력

각 단계는 `setTimeout`을 활용하여 비동기적으로 순차 실행되도록 구성하였습니다.

---

## 학습 목표

이 프로젝트를 통해 다음 개념을 학습했습니다:

- Promise 객체 생성 및 사용
- resolve / reject 흐름 제어
- async / await 문법
- then / catch 체이닝
- 비동기 함수 간 순차 실행 구조 설계
- 조건 분기를 포함한 비동기 처리

---

## 구현 핵심 개념

### Promise 기반 비동기 처리
각 기능을 Promise로 감싸고 resolve를 통해 다음 단계로 흐름을 전달했습니다.

### async / await 활용
비동기 코드를 동기 코드처럼 읽기 쉽게 구성했습니다.

### then / catch 구조
성공 흐름과 실패 흐름을 분리하여 처리했습니다.

### setTimeout을 활용한 지연 처리
비동기 실행 환경을 인위적으로 구성하여 흐름을 확인할 수 있도록 구현했습니다.
